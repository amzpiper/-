¶ 工具清单

  • Bonobo Git Server (opens new window) - Set up your own self hosted git server on IIS for Windows. Manage users and have full control over your repositories with a nice user friendly graphical interface. (Source Code (opens new window)) MIT C#
  • Fossil (opens new window) - Distributed version control system featuring wiki and bug tracker. BSD-2-Clause-FreeBSD C
  • Goodwork (opens new window) - Self hosted project management and collaboration tool powered by Laravel & VueJS. (Demo (opens new window)) MIT PHP
  • Gitblit (opens new window) - Pure Java stack for managing, viewing, and serving Git repositories. (Source Code (opens new window)) Apache-2.0 Java
  • gitbucket (opens new window) - Easily installable GitHub clone powered by Scala. (Source Code (opens new window)) Apache-2.0 Scala/Java
  • Gitea (opens new window) - Community managed fork of Gogs, lightweight code hosting solution. (Demo (opens new window), Source Code (opens new window)) MIT Go
  • GitLab (opens new window) - Self Hosted Git repository management, code reviews, issue tracking, activity feeds and wikis. (Demo (opens new window), Source Code (opens new window)) MIT Ruby
  • Gitlist (opens new window) - Web-based git repository browser - GitList allows you to browse repositories using your favorite browser, viewing files under different revisions, commit history and diffs. (Source Code (opens new window)) BSD-3-Clause PHP
  • Gitolite (opens new window) - Gitolite allows you to setup git hosting on a central server, with fine-grained access control and many more powerful features. (Source Code (opens new window)) GPL-2.0 Perl
  • GitPrep (opens new window) - Portable Github clone. (Demo (opens new window), Source Code (opens new window)) Artistic-2.0 Perl
  • Git WebUI (opens new window) - Standalone web based user interface for git repositories. Apache-2.0 Python
  • Gogs (opens new window) - Painless self-hosted Git Service written in Go. (Demo (opens new window), Source Code (opens new window)) MIT Go
  • Kallithea (opens new window) - Source code management system that supports two leading version control systems, Mercurial and Git, with a web interface. (Source Code (opens new window)) GPL-3.0 Python
  • Klaus (opens new window) - Simple, easy-to-set-up Git web viewer that Just Works. ISC Python
  • Lavagna (opens new window) - Lavagna is an open-source issue/project management tool designed for small teams. Lightweight, pure Java, easy to install, easy to use. (Source Code (opens new window)) GPL-3.0 Java
  • Octobox (opens new window) ⚠ - Take back control of your GitHub Notifications. (Source Code (opens new window)) AGPL-3.0 Ruby
  • OpenProject (opens new window) - OpenProject is a web-based project management system. (Source Code (opens new window)) GPL-3.0 Ruby
  • Phabricator (opens new window) - Collection of web applications that help build better software. (Demo (opens new window), Source Code (opens new window)) Apache-2.0 PHP
  • Phproject (opens new window) - High performance full-featured project management system. (Demo (opens new window), Source Code (opens new window)) GPL-3.0 PHP
  • ProjeQtOr (opens new window) - A complete, mature, multi-user project management system with extensive functionality for all phases of a project. (Demo (opens new window), Source Code (opens new window)) AGPL-3.0 PHP
  • Redmine (opens new window) - Redmine is a flexible project management web application. (Demo (opens new window), Source Code (opens new window)) GPL-2.0 Ruby
  • RhodeCode (opens new window) - RhodeCode is an open source platform for software development teams. It unifies and simplifies repository management for Git, Subversion, and Mercurial. (Demo (opens new window), Source Code (opens new window)) AGPL-3.0 Python
  • SCM Manager (opens new window) - The easiest way to share and manage your Git, Mercurial and Subversion repositories over http. (Source Code (opens new window)) BSD-3-Clause Java
  • Taiga (opens new window) - Agile Project Management Tool based on the Kanban and Scrum methods. (Source Code (opens new window)) AGPL-3.0 Python
  • Trac (opens new window) - Trac is an enhanced wiki and issue tracking system for software development projects. BSD-3-Clause Python
  • Tuleap (opens new window) - Tuleap is a libre suite to plan, track, code and collaborate on software projects. (Source Code (opens new window)) GPL-2.0 PHP
